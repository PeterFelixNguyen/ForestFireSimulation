This wildfire simulator is programmed in Java.
It allows the user to create maps which he can
then use to simulate the spread of wildfire. It is
primarily programmed by Peter "Felix" Nguyen.
Work on this project began when Peter was inspired
to create a more detailed Java version of Emmanuel
Medina Lopez's fire simulation coded in MATLAB.

It uses the following libraries:
SwingJD by Dhilshuk Reddy (GUI Library)
